고객만족팀 AI 업무 도구 활용 교육 - 설문 & 대시보드

## 배포 방법

### 1. GitHub에 push
```bash
git init
git add .
git commit -m "initial commit"
git remote add origin https://github.com/YOUR_USERNAME/cuckoo-ai-survey.git
git push -u origin main
```

### 2. Vercel 배포
1. vercel.com 접속 → GitHub 연결
2. Import Repository → cuckoo-ai-survey 선택
3. Framework: Vite 자동 감지
4. Deploy 클릭 → 완료
